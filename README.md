# Marketplace Farmer (Кыргызский Рынок)

Этот проект представляет собой маркетплейс для фермеров, разработанный на PHP с использованием Docker.

## Предварительные требования

Для запуска проекта вам понадобятся:
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

## Запуск проекта

1. **Сборка и запуск контейнеров:**
   Откройте терминал в корневой директории проекта и выполните команду:
   ```bash
   docker-compose up -d --build
   ```
   Эта команда соберет образ приложения и запустит контейнеры (веб-сервер Apache/PHP и базу данных MySQL) в фоновом режиме.

2. **Наполнение базы данных (Seeding):**
   После того как контейнеры запустятся, необходимо наполнить базу данных демонстрационными данными (пользователи, товары и т.д.):
   ```bash
   docker-compose exec app php seed.php
   ```

3. **Доступ к приложению:**
   Приложение будет доступно по адресу:
   [http://localhost:8081](http://localhost:8081)

## Основные команды

- **Остановка проекта:** `docker-compose down`
- **Просмотр логов:** `docker-compose logs -f`
- **Перезапуск контейнеров:** `docker-compose restart`

## Структура проекта

- `index.php` — Главная страница маркетплейса.
- `login.php` / `register.php` — Авторизация и регистрация.
- `dashboard.php` — Личный кабинет пользователя.
- `seed.php` — Скрипт для наполнения базы данных.
- `database.sql` — Схема базы данных (автоматически импортируется при первом запуске).
## Перенос на другой компьютер

### Способ 1: Через Git (Рекомендуется)
1. Создайте репозиторий на GitHub/GitLab.
2. Загрузите код:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
   git push -u origin main 
   ```
3. На другом ПК клонируйте репозиторий:
   ```bash
   git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
   ```

### Способ 2: Через архив
1. Создайте ZIP-архив папки проекта (исключая папку `db_data`, если она есть).
2. Перенесите архив на другой ПК и распакуйте.

### После переноса (на новом ПК):
1. Убедитесь, что установлен **Docker** и **Docker Compose**.
2. Запустите проект:
   ```bash
   docker-compose up -d --build
   ```
3. Наполните базу данных:
   ```bash
   docker-compose exec app php seed.php
   ```
   *Примечание: База данных будет создана автоматически при первом запуске, но она будет пустой до запуска `seed.php`.*
